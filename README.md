# Dictionary Challenge

A full-stack learning project integrating the **Dictionary API**.

Built with:

- 🔷 Next.js (Frontend)
- Playwright
- 🟩 Node.js & API routes (Backend)
- 🐘 PostgreSQL (Database)
- 📦 Prisma (ORM)
- Jest

---

## 🧪 Tech Stack

| Layer    | Tech                 |
| -------- | -------------------- |
| Frontend | Next.js              |
| Backend  | Node.js + API Routes |
| Database | PostgreSQL           |
| ORM      | Prisma               |
| Auth     | JWT                  |
